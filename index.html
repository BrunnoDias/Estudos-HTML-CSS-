Historia da internet

O que é internet ?

A internet é um sistema global de redes de computadores interconectadas que utilizam protocolos padrão para comunicação. Ela permite aos usuários acessar e compartilhar informações, comunicar-se, realizar transações comerciais e acessar uma ampla variedade de serviços, como mídia, entretenimento e educação. A internet é acessível através de dispositivos conectados, como computadores, smartphones e tablets, e é alimentada por uma variedade de tecnologias, incluindo cabo, fibra óptica e tecnologias sem fio.

Que foi a ARPANET ?

ARPANET (Advanced Research Projects Agency Network) foi a primeira rede de computadores que usou o protocolo TCP/IP, que é o protocolo padrão para a comunicação na Internet atual. Ela foi criada em 1969 pelo Departamento de Defesa dos Estados Unidos como uma rede experimental para ligar universidades e instituições de pesquisa. A ARPANET foi a base para o desenvolvimento da Internet moderna.

Como surgiu a internet no Brasil ?

Internet no Brasil surgiu nos anos 90, após a privatização das telecomunicações no país. Em 1991, a Embratel, empresa estatal de telecomunicações, criou o primeiro acesso comercial à internet no Brasil, chamado "Linha Internet". No entanto, essa oferta era bastante limitada e cara, e somente algumas empresas e instituições governamentais tinham acesso. A partir de meados da década de 90, com a abertura do mercado de telecomunicações e a chegada de novos provedores, a internet começou a se popularizar no Brasil. A partir de então, o número de usuários e a oferta de serviços foram crescendo rapidamente. Hoje em dia, a internet é amplamente utilizada em todo o país, e é considerada uma ferramenta fundamental para o desenvolvimento econômico e social.

Oque é protocolo de internet ?

Os protocolos de internet são regras e padrões que controlam como as informações são transmitidas e recebidas na internet. Eles especificam como os pacotes de dados devem ser formatados, encaminhados e recebidos, e como os erros devem ser detectados e corrigidos. Existem vários protocolos diferentes que funcionam juntos para permitir que as informações sejam transmitidas eficientemente na internet. Os protocolos mais importantes incluem:

O protocolo TCP (Transmission Control Protocol), que garante que os pacotes de dados cheguem ao destino correto e na ordem correta.
O protocolo IP (Internet Protocol), que fornece o endereçamento lógico e roteamento de pacotes de dados.
O protocolo HTTP (Hypertext Transfer Protocol), que é usado para transferir páginas da web e outros conteúdos na internet.
O protocolo HTTPS, que é uma extensão do HTTP e adiciona uma camada de segurança ao protocolo original.
O protocolo DNS (Domain Name System), que traduz nomes de domínio para endereços IP. Estes protocolos juntos, formam a base da comunicação na internet, e ajudam a garantir que as informações viajam de forma confiável e segura de um dispositivo para outro.

Cite os principais crimes cibernéticos.

Hacking: Acesso ilegal a sistemas ou redes de computadores.
Roubo de identidade: O uso não autorizado de informações pessoais para obter benefícios financeiros ou outros benefícios.
Phishing: Tentativas de obter informações pessoais ou financeiras de forma fraudulenta, geralmente através de e-mails ou mensagens de texto maliciosas.
Ransomware: Um tipo de malware que bloqueia o acesso a arquivos ou dados de um usuário e exige um resgate para desbloquear.
DDoS (Distributed Denial of Service): Ataques que visam sobrecarregar um site ou serviço com um grande volume de tráfego, tornando-o inacessível para os usuários legítimos.
Ciber-espionagem: Coleta ilegal de informações confidenciais ou de propriedade intelectual por meio de meios eletrônicos.
Fraude online: Estafas que se aproveitam da confiança dos usuários para obter dinheiro ou informações.
Invasão de privacidade: Acesso ou divulgação não autorizada de informações confidenciais ou privadas.
Propagação de malware: Dispersão intencional de programas maliciosos para danificar ou roubar informações de computadores.
Venda ilegal de informações pessoais: Venda de informações pessoais sem consentimento.

Historia da WEB

O que é a WEB ?

A World Wide Web (WWW ou Web) é um sistema de documentos interligados e hipermídia que é acessado via internet. Ele permite aos usuários encontrar e acessar informações através de links e URLs. O Web foi criado por Tim Berners-Lee em 1989 e é um dos principais componentes da internet. Ele é utilizado para acessar sites, fazer compras on-line, assistir vídeos, enviar e-mails, entre outras coisas.

Como e quando surgiu a WEB ?

A World Wide Web surgiu nos anos 80, quando o físico britânico Tim Berners-Lee, trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) na Suíça. Ele desenvolveu uma maneira de compartilhar informações entre computadores utilizando uma interface amigável para acessar documentos interligados e hipermídia.
Ele começou a desenvolver o que viria a ser conhecido como o World Wide Web em 1989, e em 1991 ele lançou a primeira versão do software que permitia acessar a Web. Ele também criou o primeiro servidor web, o primeiro navegador web, e a primeira página web, que continha informações sobre o projeto World Wide Web.

Como funciona a WEB ?

A World Wide Web funciona através de um sistema de protocolos e padrões conhecido como HTTP (Hypertext Transfer Protocol). Esse protocolo é responsável por gerenciar a comunicação entre os servidores web e os navegadores dos usuários.
Quando um usuário digita um endereço de URL (Uniform Resource Locator) em seu navegador, o navegador envia uma solicitação HTTP para o servidor web associado a esse endereço. O servidor web, por sua vez, recupera o documento solicitado e envia-o de volta ao navegador do usuário. O navegador então exibe o documento para o usuário.
Os documentos que são acessados na Web geralmente estão escritos em HTML (Hypertext Markup Language), que é uma linguagem de marcação utilizada para estruturar o conteúdo e dar formato às páginas web. Os navegadores web interpretam o HTML e exibem o conteúdo da página para o usuário.
É importante mencionar que a web é baseada no sistema cliente-servidor, onde o cliente (usuário) faz uma requisição para o servidor (computador que guarda os arquivos) e este responde com as informações solicitadas.

O que é e como surgiu HTML ?

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar páginas web. Ele foi criado por Tim Berners-Lee, um cientista da computação britânico, enquanto ele trabalhava na CERN (Organização Europeia para a Pesquisa Nuclear) na década de 1990. Berners-Lee desenvolveu HTML como uma forma de permitir que os cientistas compartilhassem documentos e informações de forma mais eficiente. O primeiro documento HTML foi publicado em 1991 e desde então, ele se tornou a base para a maioria das páginas web na internet

O que é e como surgiu CSS ?

CSS (Cascading Style Sheets) é uma linguagem de folha de estilo utilizada para descrever a apresentação de documentos escritos em uma linguagem de marcação, como HTML. Ele foi criado por Håkon Wium Lie, um engenheiro de software norueguês, no início da década de 1990. O objetivo de Lie era permitir que os desenvolvedores web pudessem separar a apresentação da estrutura do conteúdo em documentos HTML, tornando mais fácil manter e atualizar sites. O primeiro especificação do CSS foi lançada em 1996 e desde então, ele se tornou uma das principais ferramentas utilizadas para controlar a aparência de páginas web.
